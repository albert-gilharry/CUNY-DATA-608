Intro
The purpose of this assignment is to create a REST API using the Flask framework to pull data from the New York City tree census located at https://data.cityofnewyork.us/Environment/2015-Street-Tree-Census-Tree-Data/uvpi-gqnh. This data is collected by volunteers across the city, and is meant to catalog information about every single tree in the city.

I decided to create an API that will return data for trees adjacent to sidewalks. You may pull data based on two criterias:
  • Trees in which there were noticeable damages to the adjacent sidewalk flags. Data on these trees may be accessed by running the app and visiting http://localhost:5000/trees/sidewalk/Damage
  • Trees in which there were no noticeable damages to the adjacent sidewalk flags. Data on these trees may be accessed by running the app and visiting http://localhost:5000/trees/sidewalk/NoDamage
Please note that the data will be returned in JSON format. Also, there is an optional parameter that may be added to the end of the above URL to set the number of records you would like returned. The default is 1000 records. For example, http://localhost:5000/trees/sidewalk/Damage/2000 will return the 2000 trees that were adjacent to damaged sidewalks.
