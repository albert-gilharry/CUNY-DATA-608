Intro
The purpose of this assignment is to use the created REST API from Assignment 4.5. to provide visualization of choice. The API pulled data from the New York City tree census located at https://data.cityofnewyork.us/Environment/2015-Street-Tree-Census-Tree-Data/uvpi-gqnh.

I am using my API to provide a bar chart of the number of damaged or undamaged side walk near trees in each borough. You can select the integrity of the sidewalk below and be presented with the respective plot. This app uses D3.js and plotly to pull the data from the API and create the graphs. It uses Bootstrap for design. Please note the accompanying Flask APP must be running at http://localhost:5000.

For performance purposes only the first 1000 records are pulled!
